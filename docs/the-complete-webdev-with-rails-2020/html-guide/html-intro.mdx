## Let's write basic HTML!

### Create project

We are going to make project in Desktop.

In `terminal`

```bash
cd ~/Desktop
mkdir html-test # make directory
cd html-test # change directory
code . # open project in VS Code
```

### Make html file

Make `test1.html`

Copy and paste below code. I will explain the meaning of this code in the next step.

```html title="test1.html"
<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8" />
    <title>Title of the document</title>
  </head>

  <body>
    Content of the document......
  </body>
</html>
```

Refs: https://www.w3schools.com/html/html5_intro.asp
