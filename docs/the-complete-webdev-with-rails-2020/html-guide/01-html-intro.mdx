## Let's write basic HTML!

We will use an example of the below link.

### Create project

In `terminal`

```bash
mkdir html-test # make directory
cd html-test # change directory
touch test.html # make file
```

### Edit html file

```html title="test.html"
<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8" />
    <title>Title of the document</title>
  </head>

  <body>
    Content of the document......
  </body>
</html>
```

Refs: https://www.w3schools.com/html/html5_intro.asp

### Open this html file in chrome (**web browser**).

`terminal`

```bash
# under html-playground folder
open test.html
```
